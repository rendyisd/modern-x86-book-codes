# Modern x86 Assembly Programming Language Book Codes
Codes exercise (or example) from the book Modern x86 Assembly Language Programming by Daniel Kusswurm.

## Note
- The comment, code, and naming format are NOT always 1 to 1 with the book
- This code follows the System V ABI for x86-64 architecture
- Compiled to Executable and Linkable Format (ELF)
- Uses C instead of C++
- Uses NASM instead of MASM
